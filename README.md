## Installation

Clone the repository to your local development environment

```sh
https://github.com/Steve-Munoz/React-Portfolio.git
```

Run `npm install` to install all dependencies. To use the application locally, run `node server.js` in your CLI, and then open http://localhost:3000 in your preferred browswer. My Portfolio app is live on Heroku for you to view as well.
